## Usage

---
#### Clone

`https://github.com/MadeInMoon/react-awesome-timeline.git`



---
#### Install node_modules

`npm install`


---
#### Build (bundle once)

`$ npm run build-dev `


---
#### Watch (auto-bundle)

 => auto-bundle when saving file.

`$ npm run watch`
